# 坦克大战 🎮

经典 FC 风格的坦克大战游戏，使用纯 HTML5 + Canvas 实现。

## 游戏特点

- 经典像素风格图形
- 5 个精心设计的关卡
- 3 种敌方坦克类型（普通、快速、重装）
- 4 种道具（星星升级、炸弹清场、护盾无敌、铲子加固）
- 多种地形（砖墙、钢墙、河流、草地、冰面）
- 保护基地（鹰标）的经典玩法

## 操作说明

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → | 移动坦克 |
| 空格 | 射击 |
| P | 暂停/继续 |
| R | 重新开始 |

## 运行方式

直接双击 `index.html` 文件即可在浏览器中运行，无需服务器。

## 部署到 GitHub Pages

1. 将代码推送到 GitHub 仓库
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支
4. 保存后即可通过 `https://你的用户名.github.io/仓库名` 访问

## 技术栈

- HTML5 Canvas
- 原生 JavaScript
- 像素风格渲染
- 60fps 游戏循环
